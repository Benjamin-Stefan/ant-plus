Function updateState

  • updateState(sensor, state, data): void

  • Updates the state of a Bicycle Power sensor or scanner based on incoming data. Processes various types of data pages to update the sensor state, including calibration data, pedal power, cadence, and torque calculations.

    Parameters

    • sensor: BicyclePowerSensor | BicyclePowerScanner

      The sensor or scanner to update.

    • state: BicyclePowerSensorState | BicyclePowerScanState

      The state object representing the current state of the sensor.

    • data: Buffer

      The raw data buffer received from the sensor.

    Returns void

    Example

    const sensor = new BicyclePowerSensor();
    const state = new BicyclePowerSensorState(12345);
    const dataBuffer = getDataFromSensor(); // Assume this function gets data from a sensor
    updateState(sensor, state, dataBuffer);
